# TaaS SDK

Public development kit for the Friehub Truth-as-a-Service (TaaS) protocol.

## Components

### Core SDK
TypeScript library for defining and executing TaaS recipes.
- Create multi-source data ingestion logic.
- Manage execution context and state.
- Integrate with AI and search feeds.

### Protocol Prover
Compiled artifacts for ZK-based execution verification.
- Used by Challenger nodes to prove execution discrepancies on-chain.
- Succinct verification of protocol logic.

## Installation

```bash
pnpm add https://github.com/Friehub/taas-sdk
```
